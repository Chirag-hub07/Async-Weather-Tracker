# 🌦️ Async Weather Tracker

The **Async Weather Tracker** is a beginner-friendly frontend project built using **pure HTML, CSS, and Vanilla JavaScript**.
This project focuses on understanding **asynchronous JavaScript (API calls)** and **DOM manipulation** by building a real-time weather dashboard.

---

## 📌 Assignment Objective

This project helps in practicing real-world concepts like **fetching API data** and **updating UI dynamically**.

Users can:

* 🔍 Search weather by city name
* 🌡️ View real-time weather details
* 🕘 Track previously searched cities
* 📜 View logs of actions in a console panel
* 🎨 Experience a clean, styled weather dashboard

---

## 🛠️ Technology Stack

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **Vanilla JavaScript** – Async operations & DOM manipulation
* **API Used:** OpenWeatherMap

⚠️ No external libraries or frameworks are used

---

## ✨ Features

* 🌍 Real-time weather data using API
* ⚡ Asynchronous data fetching using `fetch()`
* 🧾 Dynamic UI updates using JavaScript
* 🕘 Search history buttons for quick access
* 🧹 Console log panel for tracking activity
* 🎨 Modern UI with styled weather info rows

---

## 📚 Learning Outcomes

By completing this project, you will understand:

* Asynchronous JavaScript (`fetch`, `async/await`)
* API integration in real-world apps
* DOM manipulation and dynamic content updates
* Event handling in JavaScript
* Updating UI without page reload
* Difference between `innerHTML` and `innerText`
* Debugging using browser DevTools

---

## 🚀 How to Run the Project

1. Clone or download this repository
2. Open the `index.html` file in any modern browser
3. Enter a city name and get live weather updates 🌦️
